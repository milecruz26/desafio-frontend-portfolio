# Desafio | Front-end

O objetivo deste desafio é replicar o website portfólio encontrado no figma do curso da Cubos Academy.

Elaborei a página obrigatória para aprovação do desafio, assim como as páginas opcionais. Além disso, utilizei o flexbox para tornar o layout mais responsivo dependendo do tamanho da tela.
